# Alex Painter's Homebrew Tap

A collection of Homebrew formulae for tools maintained by [@Alex-Painter](https://github.com/Alex-Painter).

## Installation

Add the tap:

```bash
brew tap Alex-Painter/tap
```

Then install any formula:

```bash
brew install <formula-name>
```

## Available Formulae

### twig

A TUI to manage git worktrees and tmux sessions.

```bash
brew install Alex-Painter/tap/twig
```

See the [twig repository](https://github.com/Alex-Painter/twig) for more information.
